# Bundle Up

## Introduction
Bundle Up is an easy-to-use weather app with clothes recommendations. It is made to be a tool especially
for parents who are short of time in the mornings. By just opening the app you will se today's weather 
along with recommended number of layers and/or special clothes for rainy days etc.
